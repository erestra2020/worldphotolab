# Hostinger Deployment Guide

## Step 1: Install WordPress
Use Hostinger hPanel > Websites > Auto Installer > WordPress.

## Step 2: Upload Theme
Upload `wordpress/themes/worldphotolab` as a ZIP from WordPress Dashboard > Appearance > Themes.

## Step 3: Upload Core Plugin
Upload `wordpress/plugins/worldphotolab-core` as a ZIP from WordPress Dashboard > Plugins > Add New.

## Step 4: Required Settings
- Enable SSL
- Set permalinks to Post Name
- Install Rank Math SEO
- Install LiteSpeed Cache
- Install Wordfence
- Configure WP Mail SMTP

## Step 5: Test
- Photographer registration
- Admin approval
- Search results
- Profile page
- Mobile responsiveness
