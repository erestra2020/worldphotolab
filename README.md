# WorldPhotoLab

Premium global photographer directory and marketplace.

## Project Goal
WorldPhotoLab is a WordPress-based platform where photographers can register, create profiles, upload profile/portfolio images, add pricing packages, and get listed publicly only after admin approval.

## Core Modules
- Public website
- Photographer registration
- Photographer dashboard
- Admin approval panel
- Searchable photographer directory
- Photographer profile pages
- Reviews and ratings
- SEO landing pages
- Future paid memberships

## Recommended Stack
- WordPress
- Custom theme: `worldphotolab`
- Custom plugin: `worldphotolab-core`
- PHP 8.2+
- MySQL/MariaDB
- Rank Math SEO
- LiteSpeed Cache
- Wordfence
- WP Mail SMTP
