# Antigravity Master Prompt

Build WorldPhotoLab.com as a premium WordPress photographer directory and marketplace.

## Main Requirements
Create a custom WordPress theme and custom plugin.

### Theme
- Premium responsive UI
- Homepage
- Photographer directory page
- Photographer profile page
- Registration page
- Blog layout
- Contact page

### Plugin
- Custom post type: Photographer
- Photographer registration form
- Admin approval workflow
- Profile status: pending, approved, rejected, suspended
- Custom fields for location, categories, pricing, experience, social links
- Search filters
- Shortcodes for directory and registration
- Admin list and approval actions

### Design
Use a premium photography style with black, white, gold, soft gradients, large imagery, clean cards, and mobile-first layout.

### Future Ready
Prepare fields for paid memberships, featured listings, verified badges, Razorpay, booking, reviews, and lead management.
