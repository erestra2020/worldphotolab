# Sprint 1 Upload Instructions

## What this package adds
- Missing docs and assets folders
- Improved WordPress custom theme
- Custom plugin for photographer post type
- Registration shortcode
- Directory shortcode
- Basic admin review workflow using WordPress pending/publish statuses

## Upload to GitHub
1. Extract this ZIP.
2. Open your GitHub repo: https://github.com/erestra2020/worldphotolab
3. Click Add file > Upload files.
4. Drag all folders/files from this package.
5. Commit message: Sprint 1 - WordPress theme and core plugin structure

## WordPress Setup
1. Install WordPress on Hostinger.
2. Create these pages:
   - Home
   - Find Photographers
   - Become a Photographer
3. Assign templates:
   - Home uses front-page automatically.
   - Find Photographers uses "Find Photographers" page template.
   - Become a Photographer uses "Become a Photographer" page template.
4. Activate WorldPhotoLab theme.
5. Activate WorldPhotoLab Core plugin.
6. Go to Settings > Permalinks > Post name > Save.
